# Read2N DAO

游戏本身是一个DAO。

WCM是Read2N的治理代币。

\


团队的成本将接受Read2N DAO的审批。每年的净收入，将由DAO决定使用。

\


游戏正式上限之后，对任何白皮书的修改，都将进行DAO投票决策。只有WCM的质押者拥有投票权。

\


质押者的投票系数和分红系数，都将根据质押时间得到加成。

| 质押时间 | 比例系数 |
| ---- | ---- |
| 1个月  | 0.5  |
| 3个月  | 0.6  |
| 6个月  | 0.7  |
| 12个月 | 1    |
| 24个月 | 1.5  |
| 36个月 | 2    |
| 60个月 | 3    |
