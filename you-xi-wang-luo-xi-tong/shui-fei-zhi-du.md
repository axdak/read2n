# 税费制度

游戏将对各种应用内活动收取不同的税费。

收费和费用的归属，和活动收费的内容相关。

\


如果该活动使用BUSD完成，

| 应用内活动 | 税费  | 分配                                                                                        |
| ----- | --- | ----------------------------------------------------------------------------------------- |
| 市场交易费 | 2%  | <p>40%项目团队用于运营发展</p><p>30%兑换为WCM支付给版权方</p><p>10%兑换为WCM支付给UGC创作者</p><p>20%兑换为WCM支付给出借者</p> |
| 市场版权费 | 4%  |                                                                                           |
| 借阅服务费 | 30% | <p>40%项目团队用于运营发展</p><p>30%兑换为WCM支付给版权方</p><p>10%兑换为WCM支付给UGC创作者</p><p>20%兑换为WCM支付给出借者</p> |
